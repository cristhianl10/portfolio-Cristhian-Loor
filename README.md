# Cristhian Loor Quimí — Portfolio

Mi landing page personal. Hecha con un colores inspirados en la sublimación.
## Stack

- React 18 + Vite
- Framer Motion (animaciones)
- Lenis (smooth scroll)
- Lucide React (iconos)
- CSS vanilla con variables

## Secciones

- **Hero** — nombre, locación, tagline, CTA
- **Sobre mí** — bio + stack técnico con marquee
- **Proyectos** — StockFlow, Literapp, Dr UX
- **Sublirium** — mi emprendimiento de sublimación
- **Contacto** — LinkedIn, GitHub, email

## Correr local

```bash
cd app/frontend
npm install
npm run dev
```

## Por qué existe esto

Para mostrar una pieza de mi portfolio. Una parte de mi profesión y desarrollo académico.
Una forma de expresar mi identidad a través de lo que me gusta
y lo que hago. La  emoción de lo cotidiano y la formación de procesos constructivos con esfuerzo y pasión.
